내가 원하는 곳들의 날씨를 찾아주는 앱 (Android Studio)
================================================================

#### 설명
내가 알고 싶은 위치들을 모아서, 그곳들의 날씨를 한번에 알 수 있도록 해주는 앱입니다.

data.go.kr에서 동네예보 조회서비스 인증키를 받아 API를 사용합니다.  
지역을 터치하면, 날씨를 설명해주는 페이지로 넘어갑니다.
